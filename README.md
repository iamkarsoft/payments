# Laravel Package Skeleton 

a scaffold for creating laravel packages



## Folder Structure

- config
	- package

- database
	- migrations

- resources
	- views ( for blade views)
	- js ( for js files )

- routes
	- web.php  web routes
	- api.php  api routes

- src 
	- Console 
	- Facades
	- Http
	- Jobs
	- Models
	- Notification
		- PackageNotification.php
	- PackageServiceProvider.php

- tests
